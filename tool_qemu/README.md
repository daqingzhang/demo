Qemu application on linux
=====================================

Website
-------------------------------------------------------------------------------
https://www.qemu.org

Download & Make
-------------------------------------------------------------------------------
https://www.qemu.org/download/

git clone git://git.qemu.org/qemu.git
cd qemu
git submodule init
git submodule update --recursive
./configure
make 


Install Qemu on ubuntu
-------------------------------------------------------------------------------
sudo apt-get install qemu
